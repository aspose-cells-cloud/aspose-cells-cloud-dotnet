# **postChartSecondValueAxis API**

 

```bash

POST http://api.aspose.cloud/v3.0//cells/{name}/worksheets/{sheetName}/charts/{chartIndex}/secondvalueaxis

```

| :- | :- | :- |:- | 
|name|String|Path||
|sheetName|String|Path||
|chartIndex|Integer|Path||
|axis|Class|Body||
|folder|String|Query||
|storageName|String|Query||
